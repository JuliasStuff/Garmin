ℹ️ About

Garmin Tracker is a personal PWA that displays watch data synced from Garmin Connect by a small Azure Function. The Function runs every hour, pulls your latest stats, sleep, and activities, and writes them to a single Firestore document. This app reads that document live and renders the views.
